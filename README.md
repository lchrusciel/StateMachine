# StateMachine

## Installation

```bash
git clone git@github.com:lchrusciel/StateMachine.git
cd StateMachine
composer install
```

## Running application

### Winzou

```bash
bin/console app:pay-with-winzou
``` 

### Workflow

```bash
bin/console app:pay-with-workflow
``` 

### Messenger

```bash
bin/console messenger:consume async
```
